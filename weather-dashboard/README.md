# Weather Dashboard

Real-time weather app using OpenWeatherMap API.

## Features

- Live weather data
- Search any city
- Auto-refresh every 5 minutes
- Loading & error states
- Clean UI with subtle colors

## Setup

1. Get free API key from openweathermap.org
2. Create `.env.local` and add: `VITE_WEATHER_API_KEY=your_key`
3. Run: `npm install && npm run dev`

## Tech Stack

React + Vite, OpenWeatherMap API, useEffect, useState
